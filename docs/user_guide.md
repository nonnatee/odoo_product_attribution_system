# Custom Product Attribution System: User Guide

Welcome to the user and administrator documentation for the **Custom Product Attribution System** Odoo module. This system enables category-based specifications, reusable attribute defaults, and a dynamic backend product template editor.

Use this guide to learn how to configure and operate the system.

## 🛠️ Access & Main Menu Navigation

Once the module is installed, its settings are integrated into standard Sales and Inventory menus. Ensure your user has access permissions.

### Navigation Paths:

* **Product Attributes**: 
  * Go to **Sales > Configuration > Products > Attributes**
  * Or go to **Inventory > Configuration > Products > Attributes**
* **Attribute Sets**: 
  * Go to **Sales > Configuration > Products > Attribute Sets**
  * Or go to **Inventory > Configuration > Products > Attribute Sets**
* **Product Categories**: 
  * Go to **Sales > Configuration > Products > Product Categories**
  * Or go to **Inventory > Configuration > Products > Product Categories**
* **Product Templates (PIM Stage Workflow & Assets)**: 
  * Go to **Sales > Products > Products**
  * Or go to **Inventory > Products > Products**
* **PIM Export Profiles**:
  * Go to **Sales > Configuration > Products > PIM Export Profiles**
  * Or go to **Inventory > Configuration > Products > PIM Export Profiles**
* **PIM App Settings**:
  * Go to **Sales > Configuration > Products > PIM Settings**
  * Or go to **Inventory > Configuration > Products > PIM Settings**
